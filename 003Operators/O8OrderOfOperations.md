# Order of Operations

- operations are performed in order of priority
- if priority is the same, operations go from left to right

| Priority | Symbol |
|--- | --- |
| 1 | ++ |
| 1 | -- |
| 1 | ()|
| 2 | ! |
| 2 | (typecast) |
| 3 | * |
| 3 | / |
| 3 | % |
| 4 | + |
| 4 | - |
| 5 | <, <= |
| 5 | >, >= |
| 6 | ==, != |
| 7 | && |
| 8 | &#124;&#124; |
| 9 | all assignment operators |
