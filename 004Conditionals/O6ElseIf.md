# Else If

```c
if (condition) {
  // Some code
} else if (condition) {
  // Some code
} else {
  // Some code
}
````
- always after if statement and before else statement
- you can have more than one else if statement

```c
if (grade == 9) {
  printf("Freshman\n");
} else if (grade == 10) {
  printf("Sophomore\n");
} else if (grade == 11) {
  printf("Junior\n");
} else if (grade == 12) {
  printf("Senior\n");
} else {
  printf("Super Senior\n");
}
```
